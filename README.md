# RSS(I) and LQI measuraments of fixed sensors for indoor WSN and autolocalization

Set of experimental RSSI measurements collected with Tmote-Sky wireless sensor nodes in different environments. You can [download]](https://github.com/r4m/rssi-data/blob/master/20101109/20101109_Automatica_Fixed_Sensor_Campaign_PUBLIC.zip) all such data [here]](https://github.com/r4m/rssi-data/blob/master/20101109/20101109_Automatica_Fixed_Sensor_Campaign_PUBLIC.zip), together with the [description](https://github.com/r4m/rssi-data/blob/master/20101109/README_20101109.txt) of the experimental settings and a short [description](https://github.com/r4m/rssi-data/blob/master/20101109/README_20101109.txt) of the platforms.
